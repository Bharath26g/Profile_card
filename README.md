# Profile_card
# Responsive Profile Card

# Project Overview
This project is a simple and responsive Profile Card built using HTML and CSS.  
It displays a user’s profile picture, name, designation, short bio, and social media links in a clean and modern card layout.  
The design is fully responsive and adapts well to different screen sizes.


# Features
- Clean and modern UI design
- Responsive layout for all screen sizes
- Circular profile image with border styling
- Hover animation for smooth user interaction
- Social media icon section
- Lightweight and fast loading


# Technologies Used
- HTML5 – Structure of the profile card
- CSS3 – Styling, layout, animations, and responsiveness
